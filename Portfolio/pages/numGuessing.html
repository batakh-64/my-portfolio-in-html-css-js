Number Guessing Game with Customizable Range

Guessing games have always been a fun way to challenge our intuition and luck. Building upon the classic number guessing concept, I have developed a web-based game that allows players to set their own range or play with the default range of 1 to 100.

Features:

  • Custom Range Selection: Players can define their own minimum and maximum values to set a personalized guessing range.
  • Default Range: If no custom range is set, the game defaults to a range between 1 and 100.
  • Interactive Feedback: After each guess, players receive feedback indicating whether their guess is too high, too low, or correct.
  • Attempt Counter: The game keeps track of the number of attempts made.

How to Play:

  1. Set the Range (Optional): Enter your desired minimum and maximum values and click "Set Range." If you prefer the default range, simply proceed without setting a custom range.
  2. Make a Guess: Enter your guessed number within the specified range.
  3. Receive Feedback: The game will inform you if your guess is too high, too low, or correct.
  4. Continue Guessing: Keep guessing until you find the correct number. The game will display the total number of attempts once you've guessed correctly.

Enjoy the game and test your guessing skills!
